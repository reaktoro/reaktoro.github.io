Map<Pair<long, long>, Cell> cells

The hash table used to access a temperature-pressure grid cell containing learned computations.

Note the use of long as number type for a temperature-pressure pairs. Temperatures and pressures are rounded to nearest checkpoints based on provided temperature/pressure step lengths for discretization.
