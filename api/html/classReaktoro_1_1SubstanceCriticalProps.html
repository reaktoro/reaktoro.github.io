const StringList &  names

Construct a SubstanceCriticalProps instance with given data.

The given names will be converted to uppercase, suffix will be removed, and spaces will be replaced by dashes. So, for example, the substance name carbon dioxide is replaced by CARBON-DIOXIDE and HCl(g) by HCL.
