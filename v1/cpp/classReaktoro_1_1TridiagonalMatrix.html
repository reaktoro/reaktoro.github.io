Detailed Description

A class that defines a Tridiagonal Matrix used on TransportSolver.

it stores data in a Eigen::VectorXd like, M = {a[0][0], a[0][1], a[0][2], a[1][0], a[1][1], a[1][2], a[2][0], a[2][1], a[2][2]}
