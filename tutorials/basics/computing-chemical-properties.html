Computing chemical properties#

Written by Allan Leal (ETH Zurich) on Jan 4th, 2022

You have now learned how to create a ChemicalSystem object and ChemicalState objects representing the chemical state of that system. We will now demonstrate how thermodynamic and chemical properties of the system, its phases, and constituting species can be calculated using class ChemicalProps.

Letâ€™s start with a simple chemical system containing a single aqueous phase:

from reaktoro import *

db = PhreeqcDatabase("phreeqc.dat")

solution = AqueousPhase(speciate("H O Na Cl C"))

system = ChemicalSystem(db, solution)

Letâ€™s create a chemical state for this system representing a 1 molal NaCl solution with 0.7 molal dissolved CO2:

state = ChemicalState(system)
state.temperature(60.0, "celsius")
state.pressure(15.0, "atm")
state.set("H2O", 1.0, "kg")
state.set("Na+", 1.0, "mol")
state.set("Cl-", 1.0, "mol")
state.set("CO2", 0.7, "mol")

equilibrate(state)

print(state)

We used the convenience method equilibrate above to quickly bring the ChemicalState object state into a state of chemical equilibrium. We will later learn how to use Reaktoroâ€™s Gibbs energy minimization algorithm to perform a wide variety of chemical equilibrium calculations using class EquilibriumSolver together with class EquilibriumSpecs, which allows us to enforce different equilibrium constraints at equilibrium (e.g., fixed pH, pE, Eh, gas fugacity, enthalpy, internal energy, volume, etc.).

We have seen that class ChemicalState is not responsible for computing thermodynamic and chemical properties of the system. Itâ€™s primary use is for preparing initial states for chemical equilibrium/kinetics calculations and also to store the result of these computations.

Computing chemical properties of ChemicalSystem objects is a task for class ChemicalProps:

By default, when defining a chemical system using AqueousPhase, GaseousPhase, LiquidPhase, SolidPhase, MineralPhase, CondensedPhase, IonExchangePhase, etc., these objects are created with ideal activity models. Thatâ€™s why you should see unit activity coefficients above for all species.

We will learn later how to specify more sophisticated and accurate activity models for each phase, but out of curiosity, we will define Pitzerâ€™s activity model from Harvie et al. [1984], Harvie and Weare [1980]. Letâ€™s also consider a gaseous phase and some mineral phases. For the gaseous phase, we shall use Peng-Robinsonâ€™s equation of state [Peng and Robinson, 1976] as its activity model.

We will need to reconstruct the ChemicalSystem object, as well as the ChemicalState and ChemicalProps objects:

solution = AqueousPhase(speciate("H O Na Cl C Ca Si"))
solution.setActivityModel(ActivityModelPitzerHMW())

gases = GaseousPhase()
gases.setActivityModel(ActivityModelPengRobinson())

minerals = MineralPhases()

system = ChemicalSystem(db, solution, gases, minerals)

state = ChemicalState(system)
state.temperature(60.0, "celsius")
state.pressure(15.0, "atm")
state.set("H2O"    , 1.0, "kg")
